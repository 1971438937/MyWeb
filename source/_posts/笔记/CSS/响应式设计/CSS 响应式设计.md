# CSS 响应式设计

## 写在前面

### 是什么

响应式设计是一种可以同时适配不同设备的网页布局方案。

同属点说，就是在不同宽度的屏幕下，都可以看到完整的页面内容，只不过，在宽度较小的屏幕下，内容可能会有删减，布局可能会有些改变等，但不会出现某些内容无法显示的情况。

描述响应式界面最著名的一句话就是 **Content is like water**，译为：如果将屏幕看作容器，那么内容就像水一样。
